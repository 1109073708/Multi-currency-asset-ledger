# Multi-currency Asset Ledger

一个轻量的多货币资产记账网页应用，用来记录美元、港币、人民币等不同货币资产，并按实时汇率折算总资产。

## 功能

- 记录银行账户资产，支持自定义银行和多币种余额
- 记录理财账户资产，支持自定义理财机构、产品名称和币种
- 支持港币、美元、人民币等资产汇总，并可切换总资产展示币种
- 自动获取实时汇率，用实时汇率进行资产折算
- 内置汇率换算器，支持双向输入金额、切换币种和交换币种
- 汇率走势图支持周、月、年切换
- 鼠标悬浮在走势图上可查看指定日期的汇率数据
- 数据保存在浏览器本地 LocalStorage，无需后端服务

## 本地运行

这是一个纯前端项目，可以直接打开 `index.html` 使用。也可以启动一个本地静态服务器：

```bash
python -m http.server 5173 --bind 127.0.0.1
```

然后在浏览器打开：

```text
http://127.0.0.1:5173/
```

## 汇率说明

应用会从 Frankfurter 汇率接口获取参考汇率，并统一折算到港币后再换算到目标展示币种。

如果网络不可用，页面会保留上一次获取到的汇率数据。

## 数据存储

当前版本的数据保存在浏览器本地：

```text
localStorage: money-ledger-v1
```

清除浏览器数据可能会导致记账数据丢失。后续可以继续增加导入导出、云同步或账户体系。
